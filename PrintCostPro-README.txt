================================================================================
                         PRINT COST PRO v1.0
                    3D Print Cost Calculator
================================================================================

This package contains two versions:

1. PrintCostPro/       - Standard version (manual entry)
2. PrintCostProPlus/   - Plus version (with STL/3MF file upload)

================================================================================
                           QUICK START
================================================================================

1. Extract this ZIP file to any folder

2. Open in your web browser:
   - Windows: Double-click index.html
   - Mac: Right-click index.html → Open With → Safari/Chrome
   - Or drag index.html into your browser window

3. That's it! No installation required.

================================================================================
                         FIRST TIME SETUP
================================================================================

1. PRINTERS TAB
   - Click "+ Add Printer"
   - Choose from 40+ presets (Bambu, Creality, Prusa, Elegoo, etc.)
   - Or enter custom printer specs
   - Set your purchase price and current hours used

2. MATERIALS TAB
   - Add your filaments/resins with current prices
   - Default materials are pre-loaded

3. SETTINGS TAB
   - Electricity rate (check your power bill, typically $0.10-0.15/kWh)
   - Labor rates for design and post-processing
   - Failure rate % (accounts for failed prints)
   - Overhead % and profit margin

================================================================================
                        CALCULATING COSTS
================================================================================

1. Go to Calculator tab
2. Select your printer and material
3. Enter from your slicer:
   - Material used (grams)
   - Print time (hours)
4. Add optional costs:
   - Hardware (bolts, screws, inserts)
   - Consumables (paint, primer, glue)
5. Set quantity and profit margin
6. View complete breakdown on the right
7. Save quote, copy to clipboard, or generate PDF

================================================================================
                    PRINT COST PRO+ FEATURES
================================================================================

The Plus version adds STL/3MF file analysis:

1. Drag & drop your 3D model file
2. App extracts: volume, dimensions, surface area, triangle count
3. Configure: infill %, wall lines, top/bottom layers
4. Click "Apply Estimation to Calculator"
5. Fine-tune values as needed

Supported files: .STL (binary & ASCII), .3MF

================================================================================
                         DATA & BACKUP
================================================================================

- All data saves automatically in your browser
- Use Export button to backup as JSON
- Use Import to restore or transfer to another computer
- Data is stored locally - nothing sent to any server
